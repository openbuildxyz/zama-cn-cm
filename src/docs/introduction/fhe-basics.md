# FHE 基础概念

全同态加密（Fully Homomorphic Encryption，FHE）是密码学领域的一项革命性技术，允许在**不解密数据**的情况下对其进行计算。

## 核心原理

传统加密方式下，数据必须先解密才能处理，这意味着服务器或第三方在计算过程中可以看到原始数据。

FHE 打破了这一限制：

```
加密数据 → FHE 计算 → 加密结果
解密结果 = 对原始数据直接计算的结果
```

## 基本操作

FHE 支持对密文（加密数据）的两种基本操作：

- **加法同态**：`Enc(a) + Enc(b) = Enc(a + b)`
- **乘法同态**：`Enc(a) × Enc(b) = Enc(a × b)`

通过组合这两种操作，可以实现任意复杂的计算。

## TFHE 方案

Zama 采用的 **TFHE（Torus FHE）** 方案具有以下特点：

- **快速自举（Fast Bootstrapping）**：支持高效的密文刷新，可执行任意深度的计算
- **门操作支持**：支持在布尔电路层面进行操作
- **低延迟**：相比其他 FHE 方案，TFHE 的单次操作延迟更低

## 噪声管理

FHE 密文中含有**噪声**，每次计算都会使噪声增大。当噪声超过阈值，解密结果将不正确。

**自举（Bootstrapping）** 操作可以刷新密文，降低噪声，使得可以执行任意深度的计算。

## 应用场景

- **隐私智能合约**：链上数据保密处理
- **私有机器学习**：在加密数据上训练/推理模型
- **医疗数据分析**：处理敏感健康数据而不暴露隐私
- **金融风控**：加密财务数据的联合计算
